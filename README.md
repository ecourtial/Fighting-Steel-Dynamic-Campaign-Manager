## Description :notebook:

An application to dynamically manage scenarios for the game _Fighting Steel_ with the _Thunder At Sea_ campaign simulator.
Development still in progress.

## Stack :light_rail:

* Symfony 5
* PHP 7.4
* PHPUnit
* Circle CI
* Infection
* PHPStan with strict rules
